# java-web-maven-spring-thyme-hello-world

## Description
A POC for springboot web app with thyme support.

## Tech stack
- java
- maven
  - springboot
  - thyme
  - bootstrap

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
[Available at](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
